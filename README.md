# Hue Twitch Bot

Let the viewers control your hue lamp with twitch chat.

#### Commands: 
- !hue blue
- !hue green
- !hue orange
- !hue pink
- !hue purple
- !hue red
- !hue white
- !hue yellow

## Setup

```
yarn install
```
Add and fill in the `.env` file.

## Run

```
yarn start
```
